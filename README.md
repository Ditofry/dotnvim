# dear neovim,

My editor setup. Plugin management is handled by [vim-plug](https://github.com/junegunn/vim-plug).

## step one

```bash
git clone https://github.com/killtheliterate/dotnvim.git ~/.config/nvim
```

## step two

Start neovim, and run `:PlugInstall`
